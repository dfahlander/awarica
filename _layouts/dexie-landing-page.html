Dexie is one of the most widely used a database libraries for apps built on modern web technology. It was developed in 2014 by David Fahlander and has been increasing in popularity ever since.

https://dexie.org














Dexie Cloud is a cloud service that makes it easy to sync data between devices and collaborate on the same data. It's built on top of Dexie.js and is designed to be easy to get started with.

https://dexie.org/cloud
